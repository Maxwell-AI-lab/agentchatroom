# Contributing to AgentChatRoom

First off, thank you for considering contributing to AgentChatRoom! 🎉

## How Can I Contribute?

### Report Bugs

- Open a [GitHub Issue](../../issues/new?template=bug_report.md)
- Include: OS, Python version, steps to reproduce, expected vs actual behavior

### Suggest Features

- Open a [GitHub Issue](../../issues/new?template=feature_request.md)
- Describe the feature, use case, and expected behavior

### Submit Pull Requests

1. Fork the repo
2. Create a feature branch: `git checkout -b feature/my-feature`
3. Make your changes
4. Ensure code quality: follow existing style
5. Commit with clear messages
6. Push and open a PR

## Development Setup

```bash
git clone https://github.com/Maxwell-AI-lab/agentchatroom.git
cd agentchatroom
pip install -r requirements.txt
cp .env.example .env  # add your API key
python app.py
```

## Code Style

- Python: Follow PEP 8, use type hints
- JS: ES6+, consistent formatting
- CSS: BEM-like naming, organized by component

## Questions?

Feel free to open an issue or start a discussion.
